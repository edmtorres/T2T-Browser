Description

Non-B DNA motifs annotation

Methods

Non-B motifs of A-phased repeats (APR), direct repeats (DR), inverted repeats (IR), mirror repeats (MR), short tandem repeats (STR) and Z-DNA (Z) were annotated using gfa (https://github.com/abcsFrederick/non-B_gfa), described in detail in: https://github.com/kxk302/non-B_gfa. G-quadruplex motifs (GQ) were annotated with Quadron (https://github.com/aleksahak/Quadron/tree/master), using a docker image found in: https://github.com/kxk302/Quadron_Docker. Quadron scores are not integers as required by bed format, so the score is rounded to the closest integer. The exact value is given in the 4th bed column.
