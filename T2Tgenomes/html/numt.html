Description

NUMT (Nuclear-mitochondrial DNA segment) annotation release 2024/10-03

Methods

NUMTs were annotated using blastn (version 2.6.0) to align mitochondrial DNA assemblies to nuclear assemblies of the same species. Overlapping NUMTs were merged using bedtools (version 2.30.0). The code can be found in Github: (https://github.com/makovalab-psu/T2T_primate_autosomes/tree/main/Detection_of_NUMTs).
