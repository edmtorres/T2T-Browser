Description

E/L Repli-seq data is a method to measure the replication timing of the genome. The data is generated by sequencing the DNA from early and late S-phase fractions. The ratio of the early to late fraction is used to determine the replication timing of the genome. The data is normalized to the genome-wide average and smoothed to remove noise. The data is represented as a log2 ratio of early to late fraction. Positive values indicate early replication, while negative values indicate late replication.

Methods

The Repli-seq data were processed in a similar way following the protocol (PMID: 29599440) and a similar pipeline available on this Github repo. The adapters of early and late fraction sequencing reads were removed using cutadapt (version 4.2) with parameters -q 0 -O 1 -m 0 and adapter sequence AGATCGGAAGAGCACACGTCTG. The reads were then mapped to the primary assembly sequence using bwa mem (version 0.7.17) with the default options. PCR duplicates of the alignment were marked and removed using the samtool rmdup tool. We further cut the genome into non-overlapping 5kb bins and counted the number of filtered reads in each bin. The read count per bins was normalized into read per kilobase per million mapped reads (RPKM), and those bins with less then 0.1 RPKM across both early and late fraction are removed. Next, we calculated the log2 ratio between the early and late fraction. The ratios were normalized using the interquartile range normalization, ie, (value - median)/IQR, where IQR represents the difference between the third quartile and the first quartile. Finally, the ratios of each chromosome are smoothed using the loess method with 300kb window size.
